Bienvenido a TransoShop, una tienda en línea donde puedes explorar una variedad de productos en diferentes categorías. Este proyecto utiliza HTML, CSS y JavaScript para crear una experiencia de compra interactiva.

---Características Principales Navegación Intuitiva: Utiliza el menú de categorías para explorar diferentes tipos de productos. Carrito de Compras: Visualiza y gestiona los productos seleccionados en tu carrito. Diseño Responsivo: La aplicación es compatible con dispositivos de diferentes tamaños, asegurando una experiencia óptima tanto en computadoras de escritorio como en dispositivos móviles.

---Tecnologías Utilizadas HTML: El lenguaje de marcado utilizado para la estructura de la página. CSS: Se utiliza para el diseño y estilo de la interfaz de usuario. JavaScript: Se implementa para la interactividad y la manipulación dinámica de datos. Bootstrap Icons: Biblioteca de iconos utilizada para mejorar la interfaz de usuario.

---Instrucciones de Uso

1-Explorar Productos:

Haz clic en las categorías del menú para explorar productos específicos. La sección principal mostrará los productos disponibles.

2-Agregar al Carrito:

Haz clic en el botón "Agregar" para añadir productos al carrito. El ícono del carrito en la esquina superior derecha mostrará el número de productos en tu carrito.

3-Ver Carrito:

Haz clic en el enlace "Carrito" para ver los productos que has agregado. Puedes eliminar productos o vaciar completamente el carrito según tus necesidades.
